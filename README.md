# Valorant Abilities Tracker

A fun React app that displays all playable Valorant agents and their abilities using the public [Valorant API](https://valorant-api.com).

## Features

- 🎯 Lists all playable Valorant agents with their portraits
- 🔍 Search agents by name
- 🏷️ Filter agents by role (Duelist, Initiator, Controller, Sentinel)
- 🖱️ Hover over any ability card to read its description
- 🎨 Valorant-themed dark UI with red accent

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- [React 19](https://react.dev/) + [Vite](https://vite.dev/)
- [Valorant API](https://valorant-api.com) – free public community API
